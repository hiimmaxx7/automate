# 随机滚动

随机滚动是指命令浏览器以完全随机的距离和速度向上或向下滚动屏幕的操作。此操作没有任何附带的配置参数。

🎥 查看更多教程视频：[在这里](https://youtu.be/cCTEtuMtz-s)。

#### 实际示例：模拟用户真实阅读行为（养号）

此操作是养号场景中极为重要的“武器”（Facebook、Pinterest、TikTok、Twitter等）。如果一个账户刚打开就立即连续点击或快速按键，社交网络的AI系统会立即将其标记为机器人并锁定账户。

* 应用方法：在使用 Go to URL 操作访问主页或详细文章后，您可以插入随机滚动操作，并与短暂的延迟命令交替使用。
* 结果：浏览器将随机上下滚动，停下来查看，然后继续滚动。这完全可以欺骗行为扫描算法，使您的 Profile 看起来像一个真实用户在阅读新闻，最大程度地提高 MMO 账户的信任度。
