# Decrease variable

在输出变量（Output variable）框中，您可以填写旧变量的相同名称以进行累减，或者填写一个完全不同的变量名称以将结果保存到新变量中，而不改变旧变量。

* Decrease by: 想要减去的数量（例如：`1`，`2`，`5`...）。
*   例如：您有变量 `$countdown = 10`。

    * 如果填写 _Output variable_ 为 `$countdown` 且 _Decrease by_ 为 `1` ➡️ 变量 `$countdown` 将减少到 `9`（累减）。
    * 如果填写 _Output variable_ 为 `$previousRow` 且 _Decrease by_ 为 `1` ➡️ 变量 `$countdown` 仍然保持为 `10`，而新变量 `$previousRow` 的值将为 `9`。
