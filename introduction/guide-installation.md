# 📥Guide to Download and Install

To get started with GPM Automate, you can easily download it from the GPM Automate homepage here.

{% embed url="https://gpmautomate.com/" %}

After downloading, run the installation file `.exe`, then click Next through the installation steps to use the software.

📌 Important note:

Please ensure that your computer has GPM Login installed and open because the Automate scripts will directly control the profiles in this browser.

You just need to double-click the `.gpmlaunch` file. At this point, the GPM Automate Runtime executable will automatically open and run the script without requiring you to perform any additional configuration.
