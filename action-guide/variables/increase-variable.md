# Increase variable

在输出变量（Output variable）框中，您可以填写旧变量的相同名称以累加值，或者填写一个完全不同的变量名称以将结果保存到新变量中，而不改变旧变量。

* Increase by: 想要增加的数量（例如：`1`，`2`，`5`...）。
*   例如：您有变量 `$row = 5`。

    * 如果填写 _Output variable_ 为 `$row` 且 _Increase by_ 为 `1` ➡️ 变量 `$row` 将增加到 `6`（累加）。
    * 如果填写 _Output variable_ 为 `$nextRow` 且 _Increase by_ 为 `1` ➡️ 变量 `$row` 仍然保持为 `5`，而新变量 `$nextRow` 的值将为 `6`。
