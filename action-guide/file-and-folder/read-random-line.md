# 文件随机读取行

文件随机读取行是一个用于打开现有的 `.txt` 文本文件的操作，自动随机抽取该文件中的一行，然后将抽取到的行内容保存到一个输出变量中。

这个操作在您管理社交媒体账户（Facebook、X、Instagram...）时非常有用，您可以随机获取一条评论（Comment）、一条状态（Status）或一个搜索关键词，以避免在各个 Profile 之间重复相同的内容。

🎥 查看更多教程视频：[在这里](https://youtu.be/EH7olDLAb9c)。

#### 配置参数：

* 文件路径：指向计算机上包含可用数据列表的 `.txt` 文件的绝对路径。
* 输出变量：保存刚刚抽取到的随机行内容的变量名称。

#### 实际示例：自动随机评论以避免垃圾信息

假设您准备好一个存储积极评论的文件，位于 `D:\GPM\comments.txt`，每条评论占一行：

```
这篇文章真不错！
太棒了。
信息非常有用，谢谢楼主。
```

当脚本导航到一篇文章并准备点击评论时，您可以按如下方式配置：

* 配置方式：
  * 调用文件随机读取行操作。
  * 文件路径：填写路径 `D:\GPM\comments.txt`。
  * 输出变量：将结果保存的变量命名为 `$randomComment`。
* 结果：每次流程运行到此步骤（或每个 Profile 独立运行时），系统将自动随机抽取上述三行中的一行（例如：Profile 1 抽中第 3 行：`"信息非常有用，谢谢楼主。"` 并赋值给变量 `$randomComment`）。您只需调用 Key press 操作将此变量传入网站的评论框即可。
