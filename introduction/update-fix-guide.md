# 🗑️ GPM Automate 卸载指南

如果 GPM Automate 工作不稳定或在更新过程中出现错误，您可能需要卸载软件并重新安装最新版本，以使系统运行更加顺畅。

### 卸载步骤

1️⃣ 步骤 1：完全关闭所有正在您计算机上打开的 GPM Automate 的 Editor 和 Runtime 窗口。

<figure><img src="../.gitbook/assets/image (10).png" alt=""><figcaption></figcaption></figure>

2️⃣ 步骤 2：在屏幕上按下 Start（Windows 图标）键，搜索 Control Panel 并打开该应用程序。

<figure><img src="../.gitbook/assets/image (11).png" alt=""><figcaption></figcaption></figure>

3️⃣ 步骤 3：在 Control Panel 窗口中，找到并选择 Uninstall a program（卸载程序）选项。

<figure><img src="../.gitbook/assets/image (12).png" alt=""><figcaption></figcaption></figure>

4️⃣ 步骤 4：在列表中找到 GPM Automate 软件，选择它并点击 Uninstall 以完全卸载。

_小提示：成功卸载后，您可以从 GPM Softwares 的主页下载最新的安装包，并像往常一样进行重新安装。_
