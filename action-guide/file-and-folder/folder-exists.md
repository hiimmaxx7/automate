# 文件是否存在

文件是否存在是一个用于检查指定路径上的文件夹是否存在于计算机硬盘上的操作。该操作的返回结果是一个逻辑值：True（如果找到文件夹）或 False（如果文件夹不存在）。

此操作通常用作在执行创建新文件夹、删除文件夹或将文件移动到其中等操作之前的缓冲步骤，帮助脚本顺利运行而不会因路径错误而突然停止。

🎥 查看更多教程视频：[在这里](https://youtu.be/EH7olDLAb9c)。

#### 配置参数：

* 文件夹路径：指向需要检查的计算机上的文件夹的绝对路径（例如：`D:\GPM_Data`）。
* 输出变量：存储检查结果的变量名称（`True` 或 `False`）。

#### 实际示例：在下载之前检查 Profile 图片存储文件夹

假设您有一个自动下载网络头像的脚本，以将其保存到每个 Profile 的专用文件夹中，结构为：`D:\GPM_Data\Profile_1`。如果文件夹 `Profile_1` 尚未在 D 盘上创建，后续的下载操作将立即出错。

为了智能地处理这一逻辑，您可以按如下配置：

* 步骤 1：调用文件是否存在操作，填写需要检查的文件夹路径为 `D:\GPM_Data\Profile_1`，将输出变量命名为 `$isFolderReady`。
* 步骤 2：使用 If 条件块进行检查：`如果 $isFolderReady = False`（意味着该文件夹尚不存在）。
* 步骤 3：在 If 块内，调用 Create folder 操作，路径为 `D:\GPM_Data\Profile_1`，以便系统自动创建此文件夹。

结果：脚本将自动进行检查。如果文件夹不存在，它将自动创建，然后再进行下载，确保运行流程始终安全且不会崩溃。
