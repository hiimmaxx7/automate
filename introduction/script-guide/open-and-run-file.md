# 🚀 3. How to open and run a .gpmlaunch file

* The fastest way: you just need to double-click directly on the `.gpmlaunch` file, and the system will automatically call the GPM Automate Runtime software to open the script.
* Centralized management: you can also open the GPM Automate Runtime software first, then click the `+ Add Application` button and select `Executable file .gpmlaunch` to find the file you want to run.
