# 停止

停止是一个用于立即中止正在运行的脚本的动作。

> 📌 注意范围：此动作仅对当前配置文件的运行线程有效，不影响同一操作中并行运行的其他线程或配置文件。

此动作通常与条件语句 If 结合使用，以处理严重错误、无法继续脚本的错误，或当线程已完成主要任务时。

#### 实际示例：当账户被 Checkpoint 时停止线程

假设您正在编写一个养号脚本。当打开浏览器时，如果发现账户已被锁定（Checkpoint），那么执行后续操作如浏览文章、点赞、发消息等完全没有意义，并且浪费机器资源。

* 配置方法：
  * 使用 If 语句检查错误消息的出现：`如果出现“账户被锁定”`。
  * 在这个 If 语句内部，您调用停止动作。
* 结果：当系统扫描到账户被锁定时，它将遇到停止命令并立即关闭该配置文件的线程。其他线程中的配置文件仍将正常运行而不受影响。
