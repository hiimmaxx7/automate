# 选择下拉菜单

选择下拉菜单是指令浏览器自动选择下拉菜单中特定值的操作。

🎥 查看更多教程视频：[在这里](https://youtu.be/cCTEtuMtz-s)。

> ⚠️ 重要注意事项：此操作仅适用于结构标准的HTML标签`<select>`（例如：选择省/城市、选择传统出生年份的菜单）。
>
> 对于由`<div>`或`<span>`标签自定义构建的新一代下拉框（如某些现代社交网络的界面），此操作将无效。在这种情况下，您需要通过以下方式处理：使用鼠标点击命令点击下拉框以展开列表 ➔ 再使用第二个鼠标点击命令点击根据该行的XPath所需选择的值。

#### 配置参数：

* XPath：直接指向网页上`<select>`标签的标识码路径（例如：`(//select)[1]`）。
* 选择文本：根据屏幕上公开显示的文本填写（例如：`<option value="VN">越南</option>` ➔ 填写值为`越南`）。

#### 实际示例：在注册账户时自动选择国家

当您的脚本运行到注册账户信息的步骤并遇到一个国家选择菜单时。

* 配置方式：
  * XPath：`(//select)[1]`
  * 选择文本：填写澳大利亚
* 结果：系统将立即激活选择框，并将屏幕上显示的值准确安全地更改为“澳大利亚”。
