# 📝 3. Input from application (配置参数由工具作者暴露)

这是使用 `.gpmlaunch` 文件时非常方便的部分。编写脚本的人在锁定文件时故意留出了一些空白单元格，以便您根据实际需求自行填写或自定义（就像您在下面的图像中看到的界面）。

* 运行工具时的注意事项：不同作者的每个 `.gpmlaunch` 文件在此部分会显示不同的数据输入框（例如：等待时间输入框、选择结果保存文件的框，或功能序号输入框...）。因此，当您打开脚本时，请注意查看这里是否有需要填写的项目，并按照工具销售者的指导进行填写。
