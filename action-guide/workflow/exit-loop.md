# 退出循环

退出循环是一个用于强制立即退出当前循环（For 或 While）的动作，无论该循环是否已运行完规定的次数。退出后，系统将继续执行位于该循环块下的操作。

此动作通常放置在 If 条件语句块中，以处理异常情况或在提前找到所需结果时使用。

#### 实际示例：在找到有效账户时停止检查

假设您使用 For 循环扫描 Excel 文件中的 100 个账户列表，以查找状态为 `"Live"` 的账户。如果第 5 行已经是 "Live" 账户，您就不需要浪费时间继续运行剩下的 95 行。

* 配置方法：
  * 在 For 块内，您使用 If 语句检查：`如果状态 = "Live"`。
  * 在该 If 块中，您处理获取账户后调用退出循环的动作。
* 结果：系统运行到第 5 行，看到条件成立，立即跳出 For 循环并继续执行下面的操作。
