# While

与 For 循环（固定次数循环）不同，While 循环通常在您不知道将要重复多少次的情况下使用，而完全依赖于运行时的实际状态。

🎥 查看更多教程视频：[在这里](https://youtu.be/4O4dPC-CNHM)。

#### 工作原理：

* 在每次循环开始之前，系统会检查您配置的条件（Condition）。
* 如果条件尚未满足，While 块中的操作将继续执行。
* 一旦条件满足，系统将停止循环并转向 While 块下方的后续操作。

#### 实际示例：等待获取 OTP 邮件/Telegram

假设您正在编写一个注册账户的脚本，需要等待系统将 OTP 代码发送到电子邮件。由于邮件到达的时间可能快或慢，您无法提前知道需要等待多久：

* 您将 Read mail code 操作放入 While 块中。
* 循环的停止条件：当变量 `$otpCode` 接收到值（不再为空）。
* 系统将持续检查并读取邮箱。一旦成功获取到 OTP 代码，条件满足，循环停止以继续执行填写代码的步骤。

> ⚠️ 重要注意事项：使用 While 循环时，您需要确保命令块中的操作会改变停止条件的状态。如果条件配置错误或邮件永远不会到达，脚本将陷入无限循环状态（脚本挂起）。
>
> 为了优化，您应该结合一个计数变量来检查次数。如果超过 10 次没有邮件，请使用 Exit loop 操作退出循环，或使用 Stop 操作完全停止程序并转到下一个 profile。
