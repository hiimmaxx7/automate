# 🚀 3. 如何打开和运行 .gpmlaunch 文件

* 最快的打开方式：您只需双击文件 `.gpmlaunch`，系统将自动调用 GPM Automate Runtime 软件来打开脚本。
* 集中管理方式：您也可以先打开 GPM Automate Runtime 软件，然后点击 `+ Add Application` 按钮，选择 `Executable file .gpmlaunch` 来找到需要运行的文件。
