# 删除文件

删除文件是一个用于完全从计算机硬盘中删除指定路径文件的操作。

🎥 查看更多教程视频：[在这里](https://youtu.be/EH7olDLAb9c)。

> ⚠️ 重要注意事项：通过此操作删除的文件将永久消失，并且不在Windows的回收站中。因此，在执行之前，您需要确保路径，以避免错误删除重要数据。

#### 配置参数：

* 文件路径：指向需要删除的计算机上的文件的绝对路径（例如：`D:\data\temp_cookie.txt`）。

#### 实际示例：在脚本结束后自动清理临时文件

在运行诸如从网上下载视频以重新上传到其他频道，或将临时数据导出到文本文件以处理字符串的脚本时。在完成工作后（视频已上传，数据已同步到Google Sheet），您需要删除这些中间文件，以避免占满计算机内存。

* 配置方法：
  * 在脚本的末尾，您调用删除文件操作。
  * 文件路径：填写已使用的临时文件的路径，例如：`D:\render\temp_video.mp4`。
* 结果：系统将清除D盘目录下的文件`temp_video.mp4`。您的工作目录将始终保持整洁，计算机在多天的自动流运行后不会出现硬盘满的情况。
