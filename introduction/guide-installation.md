# 📥下载和安装指南

要开始使用 GPM Automate，您可以轻松地在 GPM Automate 的主页上下载，点击这里。

{% embed url="https://gpmautomate.com/" %}

下载后，运行安装文件 `.exe`，然后点击 Next 通过安装步骤即可使用该软件。

📌 重要提示：

请确保您的计算机已安装并打开 GPM Login，因为 Automate 的脚本将直接控制该浏览器上的 profile。

您只需双击 `.gpmlaunch` 文件。此时，GPM Automate Runtime 执行程序将自动打开并立即运行脚本，而无需您进行任何额外的配置操作。
