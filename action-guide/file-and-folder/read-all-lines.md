# 文件读取所有行

文件读取所有行是一个用于读取文本文件 `.txt` 的操作，并自动将文件中的每一行分割成一个单独的元素，然后将所有内容存储到输出变量的数组中。

这个操作在你有一个逐行排列的数据列表（如代理列表、关键词列表、2FA 代码列表等）并希望使用循环逐行处理时非常有用。
