# 导出 Cookie

导出 Cookie 是提取当前打开网页的所有 Cookie 数据的操作，以便存储、备份或转移到其他系统、设备使用。

#### 配置参数：

* Cookie 文件：指定导出后将创建和写入数据文件的路径（例如：`D:\fb_cookie.json`）。导出的数据将自动以标准 JSON 格式打包。

#### 重要操作注意事项：

* 需要先访问 URL：您还必须导航浏览器到需要提取数据的网页，等待网页完全加载后再调用导出 Cookie 操作，以便获取当前所有的登录会话。
