# New tab

The New tab action controls the browser to automatically open an additional empty tab in the running Profile, similar to clicking the plus sign `+` on the browser's tab bar.

🎥 Watch the tutorial video: [Here](https://youtu.be/a9qNt96X1SM).

#### Practical example: Open a new tab to run tasks in parallel

When you are scraping data on a website in the current tab, but need to open another website (for example: a mail check page or a crypto price page) to gather coordinating information without interrupting the old page:

* Configuration method: Drag the New tab action into the flow.
