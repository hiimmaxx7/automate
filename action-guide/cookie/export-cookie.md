# Export cookie

Export cookie is the action of extracting all current cookie data from the open website for storage, backup, or transfer to other systems or devices.

#### Configuration parameters:

* Cookie file: The path specifying where to create and write the data file after export (e.g., `D:\fb_cookie.json`). The exported data will automatically be packaged in standard JSON format.

#### Important operational note:

* Need to Go to URL first: You must also navigate the browser to the website from which you want to retrieve data, wait for the website to fully load, and then call the Export cookie action to obtain all current login sessions.
