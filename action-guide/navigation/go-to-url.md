# Go to URL

此操作指示浏览器直接前往您指定的网页路径（URL）。

* 作用：这是工具开始工作的第一步。您只需输入网页地址，浏览器将自动进入该页面以便您执行后续步骤。

🎥 查看更多教程视频：[在这里](https://youtu.be/a9qNt96X1SM)。

#### 配置参数：

*   URL：您想要打开的网页的准确地址。

    > ⚠️ 必须注意：填写在此框中的路径必须以完整的网络协议开头，即 `https://` 或 `http://` （例如：`https://24h.com.vn`）。如果您只填写简短形式 `24h.com.vn`，系统将无法识别路由结构并报告操作错误。
* Timeout (s)：脚本等待网页加载完所有数据的时间限制（以秒为单位）。如果您留空，系统的默认值为 `60` 秒。

#### 实际示例：在使用慢速代理时调整等待时间

当您运行营销脚本或通过带有代理的 Profile 抓取数据时，有时会遇到连接速度非常慢的代理包，响应时间长或目标网页的服务器正在拥堵。如果保持默认时间，脚本可能会因超时而中断：

* 配置方法：
  * URL：`https://pinterest.com`
  * Timeout：从 `60` 改为 `120`（2分钟）。
* 结果：浏览器将耐心等待最长达 120 秒，以便网页能够加载完数据。相反，如果您的网络非常强大，并且您希望优化速度，以便在遇到死链接（die）时脚本不会挂起，您可以主动将此数字降低到 `20` 或 `30` 秒，以便系统更快地中断错误流并切换到其他账户。
