# ⚙️ 2. Yêu cầu hệ thống để mở file

* Cần cài đặt sẵn phần mềm GPM Automate (hoàn toàn MIỄN PHÍ) trên máy tính chạy hệ điều hành Windows. Hướng dẫn tải và cài đặt xem ở [mục này](../huong-dan-tai-va-cai-dat.md).
* GPM Automate sẽ tự động đăng ký đuôi file hệ thống để nhận diện định dạng `.gpmlaunch`.
