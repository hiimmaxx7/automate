# 🖥️ 1. Window & screen (窗口和显示屏)

* Window Size: 输入浏览器窗口的宽度 (X) 和高度 (Y) 的尺寸，当工具运行时（例如图中设置为 `800 x 600`）。
* Scale: 浏览器的显示缩放比例（通常默认设置为 `100%`）。
* 如果您想同时运行多个 profile（多个窗口），可以根据您的屏幕分辨率和计算机配置调整 Window Size 和 Scale 的两个参数。**简单来说，如果 Window Size 设置得越小，Scale 设置得越低（低于 100%），屏幕分辨率越高（2K, 4K），您的计算机越强（CPU Xeon, RAM 64GB,...），那么您可以同时运行更多的线程。**
* Screen: 选择您希望浏览器窗口在工具运行时显示的屏幕（适合连接多个外部显示器的用户）。
