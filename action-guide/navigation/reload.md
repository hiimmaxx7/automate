# 重新加载

重新加载是指令浏览器刷新并重新加载当前网页内容的操作，类似于您在浏览器中按下圆形箭头按钮（F5）。此操作自动执行，并且没有任何附带的配置参数。

🎥 查看更多教程视频：[在这里](https://youtu.be/a9qNt96X1SM)。

#### 实际示例：处理网页错误或暂时失去连接的故障

在自动化运行过程中，有时您会遇到网页显示错误、弹出窗口卡住无法关闭，或者网络拥堵导致界面上的某些元素无法加载的情况。

* 应用方法：将条件语句If与Wait element或Image exists操作结合使用。如果在一定时间内未找到目标元素（网页出现错误），您可以让脚本调用Reload操作。
* 结果：浏览器将重新加载页面的数据，以修复临时显示错误，帮助脚本自动克服错误并继续运行，而不会突然中断流程。
